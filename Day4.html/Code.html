Deploying a Scalable Static Website with CDN

Hosted a static website on AWS S3 and improved performance using Amazon CloudFront CDN for faster and more secure content delivery. A static website serves fixed HTML, CSS, and image files rather than generating pages on a server each time, and the CDN copies the site to locations around the world so visitors load it from a nearby server instead of one far away — keeping it fast and scalable even under heavy traffic.

Technologies used: AWS S3, CloudFront.

Implementing a Personal Cloud Storage Solution

Built a secure personal cloud storage solution using AWS S3, similar in spirit to Google Drive or Dropbox but self-built, with versioning to recover earlier file copies, IAM access control to restrict who can view or use the files, and additional data protection to guard against accidental loss.

Technologies used: AWS S3, IAM.
