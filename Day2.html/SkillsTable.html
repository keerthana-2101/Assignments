Projects

Deploying a Scalable Static Website with CDN

Hosted a static website on AWS S3 and improved performance using Amazon CloudFront CDN for faster and secure content delivery. Static website: A website with fixed pages (HTML, CSS, images) that don't change automatically — not built by a server each time. AWS S3: A place to store and show your website files online, without needing a real server. CDN (CloudFront): A system that copies your website to many locations around the world. So people far away can open your site fast, since it loads from a nearby copy instead of one far-off server. Scalable: The site can handle many visitors at once without slowing down or breaking.

Technologies used: AWS S3, CloudFront.

Implementing a Personal Cloud Storage Solution

Built a secure cloud storage solution using AWS S3 with features such as versioning, IAM access control, and data protection. Personal Cloud Storage: A place online where you can save your files (like Google Drive or Dropbox, but built by you). AWS S3: Used here as the storage space — it holds all your files safely on the internet. Versioning: Every time you update a file, S3 keeps the old copy too. So if you make a mistake, you can go back to an older version. IAM (Identity and Access Management): Controls who is allowed to see or use your files — like a lock and key system, so only you (or people you allow) can access them. Data protection: Extra safety steps to make sure your files don't get lost, deleted by accident, or accessed by the wrong person.

Technologies used: AWS S3, IAM.
